Win The Workflow Layer, Not The Hype Cycle.

이 페이지는 2026년 3월 19일 공개된 산업 특화 Agentic AI 간담회, 외부 공식 자료, 그리고 NotebookLM cross-check를 묶어 슬라이드 형식 요약과 한 장 인포그래픽으로 압축한 시각 브리프다.

Core Thesis 한국의 승부처는 foundation model 자체보다 workflow operating layer다.
Best Entry 문서 많고 승인 절차가 긴 공공·제조·금융 업무부터 시작한다.
Hard Truth 버티컬은 UI가 아니라 process data, trust layer, human escalation의 문제다.

Most Defensible Insight

Workflow Asset > Model Race

산업 특화 Agentic AI에서 가치가 쌓이는 곳은 모델 파라미터보다 문서 구조, 승인 흐름, 예외 규칙, 권한 모델, 감사 로그, human handoff 설계다.

What To Remember

  1. 처음부터 거대한 multi-agent OS를 만들지 말 것
  2. 공공 reference workflow를 수출형 자산으로 볼 것
  3. trust layer를 제품 후순위가 아니라 판매 조건으로 설계할 것

Single-Page Infographic

이 인포그래픽은 산업 특화 Agentic AI 전략을 한 장 구조로 읽히게 만든 요약본이다.

1. Strategic Core

산업 특화 Agentic AI는 모델 선택 경쟁보다 업무 흐름의 캡처와 제품화 경쟁이다. 문서 구조, 승인 단계, 예외 규칙, 권한 경계, 근거 추적이 제품 핵심이다.

Workflow Operating Layer Document-First Human Escalation Auditability

2. Where Korea Can Be Strong

공공 인허가와 규제 산업은 느린 시장이 아니라 한국형 reference workflow를 만들 수 있는 실증장이다. 이 레퍼런스는 이후 민간 확산과 해외 수출의 기준점이 될 수 있다.

3 Principles

  • workflow layer를 표준화하고 재사용하라
  • public reference workflow를 실증하고 수출하라
  • trust layer를 기본 판매 조건으로 설계하라

3 No's

  • platform-first 대규모 선행 투자 금지
  • UI-only 가짜 버티컬 금지
  • human escalation 없는 자동화 금지

Success KPI

  • 처리시간 단축
  • 재작업률 감소
  • 승인 대기시간 감소
  • 보안 사고 없음
90 Days 문서 중심 workflow 2~3개를 선정하고 baseline KPI, 근거 표시 방식, human escalation 규칙까지 포함한 pilot를 설계한다.
12 Months 도메인 스킬 카탈로그, action log, approval checkpoint, failure taxonomy를 표준 운영 패턴으로 정착시킨다.
36 Months 공공과 민간에서 검증된 reference workflow를 수출 가능한 산업 패키지로 전환하고, 국내 배치형 trust layer를 경쟁력으로 만든다.
Input 문서, 규정, process data, 예외 사유, 승인 로그
Engine LLM + tools + memory + skills + workflow graph
Guardrail source trace, permission boundary, rollback, human handoff
Output 더 빠른 처리, 더 낮은 재작업, 더 높은 신뢰, 더 강한 수출 가능성
